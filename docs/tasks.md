# 个人网站任务清单

## 项目概览
- **项目名称**：个人网站
- **类型**：单页滚动展示网站
- **核心区域**：建模区、绘图区、宠物区
- **视觉风格**：建模区(冰冷机械化) + 绘图区/宠物区(温暖文艺)

## 注意事项

1. **图片准备**：建议在开发前先准备好所有图片，统一命名和压缩
2. **样式隔离**：建模区样式和温暖区样式要严格隔离，避免冲突
3. **性能优先**：图片懒加载是核心，确保首屏加载速度
4. **移动优先**：响应式设计采用移动优先策略
5. **测试充分**：在不同设备和浏览器上充分测试
