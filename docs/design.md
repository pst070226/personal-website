# 个人网站设计文档

## 1. 项目概述

### 1.1 项目定位
一个个人项目展示网站，兼具博客分享性质，用于展示和分享用户的建模作品、绘图作品和宠物相关内容。

### 1.2 目标用户
- 访问者：对建模、绘图、宠物感兴趣的人群
- 潜在雇主或合作方
- 朋友和社交圈

### 1.3 核心价值
- 展示个人创作能力
- 分享个人兴趣和生活方式
- 建立个人品牌形象

---

## 2. 信息架构

### 2.1 网站结构

```
单页滚动网站
├── 顶部导航栏（固定）
│   ├── Logo / 个人名称
│   ├── 建模区
│   ├── 绘图区
│   └── 宠物区
│
├── Hero 区域（首页顶部）
│   ├── 个人简介
│   └── 背景氛围图
│
├── 建模区 (#modeling)
│   ├── 区域标题
│   ├── 区域简介
│   └── 作品展示网格
│
├── 绘图区 (#drawing)
│   ├── 区域标题
│   ├── 区域简介
│   └── 作品展示网格
│
├── 宠物区 (#pets)
│   ├── 区域标题
│   ├── 区域简介
│   └── 宠物照片/故事
│
└── 页脚
    ├── 联系方式
    └── 社交媒体链接
```

### 2.2 导航机制
- 单页滚动设计
- 导航栏固定在顶部
- 点击导航项平滑滚动到对应区域
- 当前区域高亮显示

## 3. 视觉设计

### 3.1 设计理念
**冰火对比** - 建模区展现冰冷的机械化美学，绘图区和宠物区呈现温暖文艺的氛围。通过视觉对比增强各区域的个性表达。

### 3.2 整体色调

#### 建模区 - 机械化风格
```
主色调：
- 深空黑: #0a0a0f
- 钢铁灰: #1a1a24
- 金属银: #2a2a3a

强调色：
- 冰蓝: #00d4ff
- 电光青: #00ffcc
- 霓虹紫: #a855f7

效果：
- 细线边框（发光效果）
- 网格背景
- 扫描线纹理
- 轻微噪点
```

#### 绘图区 & 宠物区 - 温暖文艺风格
```
主色调（Morandi 色系）：
- 暖米白: #f5f0e8
- 柔粉灰: #d4c4b5
- 莫兰迪绿: #8fa88f
- 淡陶土: #c9a88a

强调色：
- 暖橙: #e8a87c
- 柔粉: #c9b1b1
- 深棕: #5d4e3c

效果：
- 柔和阴影
- 圆润边角
- 纸张纹理（可选）
- 温暖渐变
```

### 3.3 字体规范

#### 建模区
```css
标题字体: 'Orbitron', 'Rajdhani', monospace
正文字体: 'Roboto Mono', 'Source Code Pro', monospace

特点：
- 几何感强
- 等宽字体
- 锐利边缘
```

#### 绘图区 & 宠物区
```css
标题字体: 'Playfair Display', 'Cormorant Garamond', serif
正文字体: 'Noto Sans SC', 'Source Han Sans', sans-serif

特点：
- 衬线优雅
- 圆润友好
- 人文气质
```

### 3.4 间距系统
```
基础单位: 8px

间距等级:
- xs: 8px
- sm: 16px
- md: 24px
- lg: 32px
- xl: 48px
- xxl: 64px
- xxxl: 96px

区域间距: 96px - 120px（大留白，呼吸感）
```

## 4. 组件设计

### 4.1 导航栏

```
位置：固定顶部
高度：64px
背景：半透明毛玻璃效果
行为：
- 向下滚动时略微缩小
- 当前区域导航项高亮
- 移动端折叠为汉堡菜单
```

### 4.2 Hero 区域

```
高度：100vh（视窗高度）
内容：
- 个人名称（大标题）
- 一句话简介
- 滚动提示箭头
背景：
- 微妙的渐变或纹理
- 可选：动态粒子效果（建模风格过渡）
```

### 4.3 作品卡片

#### 建模区卡片
```
结构：
┌─────────────────────┐
│ ┌─────────────────┐ │
│ │                 │ │
│ │   作品预览图    │ │
│ │                 │ │
│ └─────────────────┘ │
│ ─────────────────── │ ← 细线分隔
│ 作品名称           │
│ 技术标签           │
└─────────────────────┘

样式：
- 深色背景
- 细线边框（带发光）
- 悬停时边框变亮
- 金属质感
```

#### 绘图区/宠物区卡片
```
结构：
┌─────────────────────┐
│ ┌─────────────────┐ │
│ │                 │ │
│ │   作品/照片    │ │
│ │                 │ │
│ └─────────────────┘ │
│                     │
│ 作品名称           │
│ 简短描述           │
└─────────────────────┘

样式：
- 浅色/米色背景
- 柔和阴影
- 圆角设计
- 悬停时轻微上浮
- 温暖质感
```

### 4.4 区域过渡

```
建模区 → 绘图区：
- 渐变过渡
- 冷色调逐渐变暖
- 可能加入过渡动画

绘图区 → 宠物区：
- 保持温暖色调
- 平滑衔接
```

## 5. 交互设计

### 5.1 滚动交互
- 平滑滚动（smooth scroll）
- 区域进入时的渐入动画
- 导航栏高亮跟随

### 5.2 卡片交互
- 悬停：轻微放大 + 阴影增强
- 点击：可选放大预览或跳转详情
- 触摸设备：轻触显示信息

### 5.3 导航交互
- 点击：平滑滚动到目标
- 当前区域：高亮指示
- 移动端：汉堡菜单展开/收起

### 5.4 图片展示
- 懒加载：滚动到可视区域时加载
- 占位符：加载时显示模糊缩略图或骨架屏
- 错误处理：显示优雅的错误占位图

## 6. 响应式设计

### 6.1 断点定义
```
移动端: < 768px
平板: 768px - 1024px
桌面: > 1024px
大屏: > 1440px
```

### 6.2 布局适配

#### 移动端 (< 768px)
```
导航：
- 折叠为汉堡菜单
- 全宽覆盖式菜单

卡片网格：
- 单列布局
- 卡片全宽

间距：
- 减小区域间距
- 紧凑的内边距
```

#### 平板 (768px - 1024px)
```
卡片网格：
- 双列布局

间距：
- 中等区域间距
```

#### 桌面 (> 1024px)
```
卡片网格：
- 建模区：3列
- 绘图区：3列
- 宠物区：2-3列（不规则网格可选）

间距：
- 完整区域间距
- 最大内容宽度：1200px - 1400px
```

## 8. 文件命名规范

### 8.1 图片命名
```
格式：{区域}-{类型}-{序号}-{描述}.{扩展名}

示例：
- modeling-render-01-character.jpg     # 建模渲染-角色
- modeling-wireframe-02-vehicle.png     # 建模线框-载具
- drawing-illustration-01-portrait.jpg  # 绘图插画-肖像
- drawing-sketch-02-landscape.jpg       # 绘图素描-风景
- pets-photo-01-cat-sleeping.jpg        # 宠物照片-猫睡觉
- pets-photo-02-dog-playing.jpg         # 宠物照片-狗玩耍
```

### 8.2 组件命名
```
PascalCase 命名
- Navbar.vue
- ModelCard.vue
- DrawingSection.vue
```
